# Backdrop\Contracts
Standalone package that provides general usage for the Backdrop Framework.

## Requirements
* ClassicPress 1.4+
* WordPress 4.9+
* PHP 7.1+
* [Composer](https://getcomposer.org) for managing PHP dependencies.
